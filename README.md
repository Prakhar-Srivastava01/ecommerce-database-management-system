# E-Commerce Database Management System

## Overview

This project is a MySQL-based E-Commerce Database Management System designed to manage products,
suppliers, categories, orders, and pricing information.

## Features

* Product Management
* Supplier Management
* Category Management
* Order Management
* Product Price Tracking
* JSON-based Product Metadata
* Relational Database Design
* SQL Queries with Joins and Indexing

## Database Tables

* Supplier
* Category
* Product
* ProductPrice
* Orders
* OrderDetail

## Technologies Used

* MySQL
* SQL
* phpMyAdmin

## Key Concepts Implemented

* Primary Keys
* Foreign Keys
* Indexes
* JSON Data Storage
* CRUD Operations
* SQL Joins
* Database Backup & Export

## Project Structure

* `ecommercedb.sql` – Complete database schema and sample data
